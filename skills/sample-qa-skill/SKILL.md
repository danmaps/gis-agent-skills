# Sample QA Skill

A tiny example skill that answers a question succinctly.

## Inputs

- `question` (string)

## Output

- `answer` (string)

## Example

**User:** “What is a geodatabase?”

**Output:** “A geodatabase is Esri’s container format for storing spatial data, tables, and relationships.”
